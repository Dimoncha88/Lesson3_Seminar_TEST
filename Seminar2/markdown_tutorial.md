# Туториал по языку разметки Markdown #

## Работа с заголовками ##

Количество символов "#" указыает на уровень заголовка. Пример:

Один символ "#" перед текстом -  Заголовок первого уровня h1 
# Заголовок h1

"##" Заголовок h2
## Заголовок h2

"###" Заголовок h3

### Заголовок h3
"####" Заголовок h4
#### Заголовок h4
